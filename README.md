# Outsource Dashboard

Управление сотрудниками и проектами с финансовыми расчетами.

## 🚀 Функционал

- **Проекты**: добавление, удаление, сортировка, фильтрация
- **Сотрудники**: CRUD, inline-редактирование зарплаты/должности
- **Назначения**: привязка сотрудников к проектам с расчетом эффективности (Capacity × Fit)
- **Отпуска**: календарь доступности, влияющий на финансовые показатели
- **Финансы**: расчет Revenue, Cost, Profit с учетом коэффициента отпуска
- **Seed Data**: копирование данных между месяцами
